About GanttWarrior

GanttWarrior is a terminal-based project scheduler built for developers who live in the CLI. It combines Gantt charts, Kanban boards, and calendar views in a single TUI application — no browser, no Electron, no subscriptions.

Built with Textual and Rich, GanttWarrior runs anywhere Python runs.

Open Source

GanttWarrior is released under the MIT License. Contributions welcome.
