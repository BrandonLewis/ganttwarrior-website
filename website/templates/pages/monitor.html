how to run · WebUSB requires HTTPS or localhost (file:// is hit-or-miss). From the directory containing this file: python3 -m http.server 8000  then open http://localhost:8000/el-usb-tc-spike.html in Chrome / Edge.
first run · plug in the EL-USB-TC, click Connect, pick the device from the Chrome dialog, click Load Config, then Download Samples. On Linux you may need a udev rule: SUBSYSTEM=="usb",ATTRS{idVendor}=="10c4",ATTRS{idProduct}=="0002",MODE="0666" in /etc/udev/rules.d/99-lascar.rules then sudo udevadm control --reload.
sample decoder · the sigrok wiki documents the 3-command protocol but not the per-sample byte format for the -TC variant. The dropdown lets you swap candidates; the most common pattern for thermocouple loggers in this family is signed 16-bit little-endian × 0.1.
