# Keyboard Shortcuts

## Global

| Key | Action |
|-----|--------|
| Tab | Switch between Gantt, Kanban, Calendar |
| Escape | Focus tab bar / exit current mode |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+S | Save project |

## Gantt Chart

| Key | Action |
|-----|--------|
| Arrow keys | Move cursor |
| Shift+Arrow | Extend selection |
| f | Fill selected cells |
| d | Delete selected cells |
| c | Copy selection |
| x | Cut selection |
| p | Paste clipboard |

## Kanban Board

| Key | Action |
|-----|--------|
| Arrow keys | Navigate cards |
| Enter | Edit selected card |
| h / l | Move card left/right (change status) |
