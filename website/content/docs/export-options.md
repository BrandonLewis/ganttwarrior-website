# Export Options

GanttWarrior supports multiple export formats.

## PDF

    :::bash
    ganttwarrior export pdf schedule.pdf

Generates a formatted PDF with the Gantt chart, task list, and dependency information.

## Excel

    :::bash
    ganttwarrior export excel schedule.xlsx

Exports tasks, dates, and dependencies to an Excel spreadsheet.

## Terminal

    :::bash
    ganttwarrior export terminal

Prints a formatted Gantt chart directly to the terminal using Rich.
