# Installation

## Requirements

- Python 3.10 or later
- A terminal with 256-color support (most modern terminals)

## Install from PyPI

    :::bash
    pip install ganttwarrior

## Install from Source

    :::bash
    git clone https://github.com/BrandonLewis/ganttwarrior.git
    cd ganttwarrior
    pip install -e .

## Verify Installation

    :::bash
    ganttwarrior --version
